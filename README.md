# Resume UI using HTML

## Description
This project is a simple and clean Resume User Interface created using semantic HTML.
The main focus is on proper structure, spacing, alignment, and readability.

## Technologies Used
- HTML5

## Folder Structure
RESUME/

│── README.md

│ └── images/

## Setup Steps
1. Clone the repository
2. Open `index.html` in any web browser

## Usage
- View the resume layout
- Understand semantic HTML tags
- Observe clean UI structure and alignment
